# LinkedIn Competitor Monitoring and Connection Request Script (Windows)

This script allows you to monitor the new connections of your competitors' decision-makers on LinkedIn and send personalized connection requests based on their profiles. Please note that scraping LinkedIn or sending automated connection requests may violate LinkedIn's terms of use and ethical guidelines. Use this script responsibly and ensure compliance with all relevant rules and regulations.

## Prerequisites

- Python 3.x installed ([Download Python](https://www.python.org/downloads/))
- Chrome web browser installed ([Download Chrome](https://www.google.com/chrome/))
- ChromeDriver executable ([Download ChromeDriver](https://sites.google.com/chromium.org/driver/))

## Setup

1. **Clone the Repository:**

   ```bash
   git clone https://github.com/prasannakumarg0609/linkedin-competitor-monitor.git
   cd linkedin-competitor-monitor

2. Install Dependencies:

pip install beautifulsoup4
pip install requests
pip install selenium



3. Download and Extract ChromeDriver:

Download the appropriate version of ChromeDriver that matches your Chrome browser version from the ChromeDriver download page. Place the downloaded chromedriver.exe executable in the project directory.

## Usage

Run the script(main.py) in VS code --> run --> start debugging
